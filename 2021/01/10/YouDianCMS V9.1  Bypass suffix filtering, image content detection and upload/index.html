YouDianCMS V9.1 Bypass suffix filtering, image content detection and upload

Vulnerability Type :

File upload

Vulnerability Version :

YouDianCMS V9.1

Recurring environment:

  • Windows 10
  • PHP 5.4.5
  • Apache 2.4.23

Vulnerability Description AND recurrence:

Vulnerability hazard

Through this vulnerability, we can bypass the restriction of upload suffix and the detection of image content, and upload PHP files that can execute commands, resulting in the server being controlled by hackers!

Causes of loopholes

The vulnerability file is in the\ BaseAction.class.php
